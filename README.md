# Java Swing 二进制计算器

一个简易的二进制计算器，支持所有基本运算（二进制加法、减法、与、或、异或、取反）和求补运算，并且支持不同字长（8 位、16 位、32 位），具备合理的输入错误提示。

### 按键功能说明

|  按钮   |       功能       |       备注       |
| :-----: | :--------------: | :--------------: |
|   `+`   |        加        |        -         |
|   `-`   |        减        |        -         |
|   `&`   |        与        |        -         |
|  `\|`   |        或        |        -         |
|   `^`   |       异或       |        -         |
|   `~`   |       取反       | 只接受一个操作数 |
|   `#`   |       求补       | 只接受一个操作数 |
| `clear` |     清除屏幕     |        -         |
|   `1`   |     输入 `1`     |        -         |
|   `0`   |     输入 `0`     |        -         |
|  `ans`  | 输入上次计算结果 |        -         |
|   `=`   |     计算结果     |        -         |
